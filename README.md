# 🦖 Automated Dino Game

An automated bot that plays the Google Chrome Dinosaur Game on its own using Python.  
The project detects obstacles on the screen and triggers jump actions automatically to keep the dinosaur alive.

---

## 🚀 Features

- 🎮 Fully automated gameplay
- 👁️ Real-time obstacle detection
- ⚡ Fast response using Python automation
- 🧠 Simple logic, beginner-friendly project
- 🖥️ Works with Chrome Dino (`chrome://dino` or online version)

---

## 🛠️ Tech Stack

- Python
- OpenCV (optional for detection)
- PyAutoGUI
- NumPy
